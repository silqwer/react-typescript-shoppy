# Shoppy

### Page 구성

```
/ 👉 `<Home>`
/products 👉 `<AllProducts>`
/products/new 👉 `<NewProducts>`
/products/:id 👉 `<ProductDetail>`
/carts 👉 `<MyCart>`
/notfound 👉 `<NotFound>`
```

### 개발 스택

- react

- react-router-dom

- react-query

- react-icons

- typescript

- axios

- tailwind

### 환경변수

- REACT_APP_API_KEY=api-key

- REACT_APP_AUTH_DOMAIN=auth-domain

- REACT_APP_DATABASE_URL=database-url

- REACT_APP_PROJECT_ID=project-id
